- To show contents of ls_com.txt file, type "cat ls_com.txt".
- To generate ls_out.txt file, type "ls -h -s ~ > ls_out.txt".
- To compile hello.c, type "gcc -Wall -std=c99 hello.c -o hello".
- To run the hello file, type "./hello".
- To create the tar file, type "tar cvf submit.tar ls_com.txt ls_out.txt hello.c
- To create a test directory, type "mkdir test". 
- To copy tar file to test directory, type "cp submit.tar test" (or type "mv" instead of "cp" if you don't want to copy but only want to move).
- To untar file, cd into test directory and type "tar xvf sumbit.tar".
